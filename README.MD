# questionnaire-renderer

### Instructions
- #### Installation
  - Install `npm install`
  - Install **gulp** - `npm install -g gulp`
- #### Start Application
    - In Development Mode      
      - `gulp`
      - `npm start`
    - In Production Mode
      - `gulp --env=production`
      - `npm start -- --env=production`
- #### Validate Code Quality
    - `gulp buildALL`
